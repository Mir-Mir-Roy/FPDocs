Shortcuts and usage

Pivot Gizmo:

  1. This visual control gives ability to set Orientation and Origin freely.
  2. 'P' button in left bottom corner switches On/Off Pivot gizmo.

Main action:

  1. Press D + LMB - To set just Transform Orientation.
  2. Press D + MMB - Sets only 3D Cursor to highlighted.
  3. Press D + RMB - Sets only Origin to highlighted.

Additional shortcuts with main action gives some options:

  1. Press D + S + LMB or RMB - Sets Orientation and Origin.
  2. Press D + C + RMB - Sets 3D Cursor (this suits if 2 button mouse).

Additional shortcuts while holding D key:

  1. Press O - Switch On/Off Pivot to Cursor settings.
  2. Press P - Switch On/Off SCPO into Active Transform Tools settings.

Multiselection mode:

  1. Press D + A + MMB or RMB - Activate multi selection mode for 3D Cursor or Origin.
  2. If release A - Set(confirm) Cursor or Origin between selection.
  3. If release D - Exit the mode without confirmation.

One axel mode:

  1. Press D + X or Y or Z + RMB - Set Origin to one axis.

Bounding Box mode:

  1. Press D + B - Activate Bounding Box mode and widget.
  2. Press D + B + MMB or RMB - Sets 3D Cursor or Origin to bbox point.
  3. 'L' button in left bottom corner sets bounding box to local\world orientation.
    !!!Warning!!! - World orientation('W') is heavy calculation with hipoly meshes
